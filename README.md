# De Open Brouwerij
Website voor openbrouwerij.nl

In de Open Brouwerij broouw je zelf het bier!

## Requirements

- [npm](https://www.npmjs.com/)
- [Bundler](http://bundler.io/)

## Install

>```bash
gem install bundler
npm install
>```

## Running locally

>```bash
npm start
>```

## Deploying

This will deploy to GitHub pages:

>```bash
npm run deploy
>```
